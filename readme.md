## Installation
- Install library 
```
composer install
npm install
```

- Generate key 
```
php artisan key:generate
```

- Setting environment variables in .env 
- Migrate database
```
php artisan migrate
php artisan db:seed
```

- Bundle asset file 
```
npm run dev
```
